Измените предложенную программу, чтобы она выводила:

```
row1
row2
row3
row4
```

В программе сделаны самые частые ошибки начинающего программиста. 

```go
package main

import "fmt"

func main(){                             
     Fmt.Print("row1")
     fmt.Println(row2)   
     fmt.println("row3")
     fmt.PrintLn('row4')
}
```
